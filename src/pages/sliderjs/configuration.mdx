# Configuration

You can customize the slider using the following attributes:

```html
<div class="sliderjs" 
     data-speed="1000" 
     data-autoplay="false" 
     data-loop="false" 
     data-nav="false" 
     data-dots="false" 
     data-counter="false" 
     data-control="false" 
     data-overlay="true">
</div>
```
